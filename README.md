# üìö Publications & Research

* **[Exploring Extractive and Abstractive Approaches for Myanmar Language Text Summarization](https://ieeexplore.ieee.org/document/10754935)**
    <br><i>Published in IEEE Xplore</i>
    <br>Comparatively analyzed TextRank (extractive) and mT5 (abstractive) models on the XL-Sum dataset. Results demonstrated that the extractive approach achieved a superior ROUGE-1 score of 43.14, outperforming the abstractive model (36.67) and establishing benchmarks for low-resource language processing.

* **[Systematic Comparison of Vectorization Methods in Topic Classification for Myanmar Language](https://ieeexplore.ieee.org/document/11011194)**
    <br><i>Published in IEEE Xplore</i>
    <br>Evaluated TF-IDF, Count Vectorizer, and Word2Vec algorithms for news classification using an RBF-SVM model. Demonstrated that TF-IDF achieved the highest accuracy (83%) after Chi-Square feature selection, significantly outperforming Word2Vec (48%).

* **[Covid-19 Vaccine Tweets Sentiment Analysis Using Apache Spark and Mongo DB](https://www.researchgate.net/publication/393664331_Covid-19_Vaccine_Tweets_Sentiment_Analysis_Using_Apache_Spark_and_Mongo_DB)**
